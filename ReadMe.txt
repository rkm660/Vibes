Things you’ll need to install to run this:

1. Cordova — http://ngcordova.com/docs/install/

“bower install —-save-dev ngCordova”

2. Geolocation Plugin — http://ngcordova.com/docs/plugins/geolocation/

“cordova plugin add cordova-plugin-geolocation”


3. Background Geolocation Plugin — http://ngcordova.com/docs/plugins/backgroundGeolocation/

“cordova plugin add https://github.com/transistorsoft/cordova-background-geolocation-lt”  NOT the one they say on the website
